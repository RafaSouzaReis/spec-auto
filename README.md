# spec-auto

Projeto configurado com Next.js (JavaScript) e alias de import absoluto.

Tailwind CSS ja esta configurado via PostCSS para uso imediato.

## Instalar dependencias

```bash
bun install
```

## Rodar em desenvolvimento

```bash
bun run dev
```

## Build de producao

```bash
bun run build
```

## Alias de import absoluto

Use `@/` para importar a partir de `src`.

Exemplo:

```js
import HelloCard from "@/components/HelloCard";
```
